# DevOps Mini Project: Flask + Redis

## 🔧 Stack
- Flask
- Redis
- Docker
- Docker Compose
- GitHub Actions (CI/CD)

## ▶️ How to Run

```bash
docker-compose up --build
```

Visit: [http://localhost:5000](http://localhost:5000)

## 🛠️ CI/CD

GitHub Actions will automatically build the app on every push to `main` branch.
